Modèle OSI

```

Définition

Le modèle OSI (Open Systems Interconnection) est un modèle de référence qui décrit comment les données circulent à travers un réseau. Il est composé de 7 couches permettant de comprendre et de dépanner les communications réseau.

```
```

Les 7 couches OSI

Couche Nom Exemple
7 Application HTTP, DNS, FTP
6 Présentation SSL, TLS, Encodage
5 Session Gestion des sessions
4 Transport TCP, UDP
3 Réseau IPv4, IPv6, OSPF
2 Liaison de données Ethernet, MAC
1 Physique Câbles, Fibre, Wi-Fi
```
```

Équipements réseau

  • Routeur : Couche 3
  • Switch : Couche 2
  • Hub : Couche 1
```
```

Encapsulation

Lorsqu'un appareil envoie des données, chaque couche ajoute ses propres informations avant la transmission. Ce processus est appelé encapsulation.

À la réception, les informations sont retirées couche par couche : c'est la désencapsulation.

```
```

Points clés à retenir

  • Le modèle OSI possède 7 couches.
  • Chaque couche a un rôle spécifique.
  • TCP fonctionne à la couche 4.
  • IP fonctionne à la couche 3.
  • Les adresses MAC sont utilisées à la couche 2.
  • Le modèle OSI est un outil essentiel pour le dépannage réseau.
```
